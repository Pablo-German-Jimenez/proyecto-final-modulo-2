# Imágenes del Carrusel

Esta carpeta contiene las imágenes de fondo para el carrusel del HeroMovie.

## Imágenes necesarias:
- dune.jpg
- oppenheimer.jpg  
- barbie.jpg
- spiderman.jpg
- avatar.jpg

## Notas:
- Las imágenes deben tener una resolución alta (mínimo 1920x1080)
- Formato recomendado: JPG o PNG
- Tamaño recomendado: entre 500KB y 2MB por imagen
- Las imágenes se mostrarán como fondo con overlay oscuro para mejor legibilidad del texto
